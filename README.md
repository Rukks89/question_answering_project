# question_answering_project
This project is about Machine Learning-based Question Answering System built with Transformers (BERT) and a web interface using Streamlit. It allows users to input a paragraph and ask questions, and the model provides accurate answers extracted from the paragraph.


# Features
>Input any paragraph of text.

>Ask questions related to the paragraph.

>Get accurate, contextual answers using a pre-trained BERT model.

>Clean and simple Streamlit-based web interface.
